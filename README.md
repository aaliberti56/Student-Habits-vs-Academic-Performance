# Student Habits vs Academic Performance

## Obiettivo

Analizzare l'impatto delle abitudini degli studenti sul rendimento accademico
mediante:

- Analisi descrittiva
- Correlazione di Pearson
- Analisi delle Componenti Principali (PCA)
- Regressione Lineare Multipla

## Dataset

Dataset: Student Habits vs Academic Performance

Numero osservazioni: 1000

Numero variabili: 16

## Struttura Repository

- relazione/: relazione completa del progetto
- presentazione/: slide utilizzate per la presentazione
- codice_R/: script R utilizzati nell'analisi
- grafici/: grafici prodotti durante l'analisi
- dati/: dataset originale
- risultati/: output statistici

## Software utilizzato

- R
- RStudio

## Principali risultati

- Le ore di studio sono il fattore più influente sul punteggio d'esame.
- Social media e streaming mostrano effetti negativi.
- Sonno, attività fisica e benessere mentale hanno effetti positivi.
- Il modello finale spiega circa il 90% della variabilità del rendimento accademico.
